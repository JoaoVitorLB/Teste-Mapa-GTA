# GTA V Race Planner 🏁

Uma ferramenta web interativa para planejar e visualizar rotas de corrida no GTA V.

## ✨ Funcionalidades
- **Mapa Interativo**: Navegue pelo mapa oficial do GTA V utilizando Leaflet.js.
- **Checkpoints Customizados**: Clique em qualquer lugar do mapa para adicionar um checkpoint.
- **Traçado de Rota**: O sistema desenha automaticamente o percurso entre os pontos marcados.
- **Gerenciamento em Tempo Real**: Remova, arraste ou limpe todos os checkpoints facilmente pelo painel lateral.
- **Exportação**: Baixe seu plano de corrida em formato JSON para referência futura.
- **Responsivo**: Design moderno que se adapta a diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3**: Estrutura e estilização premium com foco em experiência do usuário.
- **JavaScript (ES6+)**: Lógica de manipulação de dados e DOM.
- **Leaflet.js**: Motor de mapa interativo.
- **Lucide Icons**: Conjunto de ícones vetoriais modernos.

## 🚀 Como Executar
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para hospedagem, ative o **GitHub Pages** nas configurações do seu repositório.
